# Estetica Medica - Dra. Szachraj Casco Mayra

Landing page profesional para la Dra. Szachraj Casco Mayra, enfocada en estetica medica, tratamientos faciales y corporales, turnos por WhatsApp y ubicacion del consultorio en Corrientes Capital.

## Stack

- Next.js
- React
- Tailwind CSS
- Netlify

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir:

```text
http://localhost:3000
```

## Build

```bash
npm run build
```

## Deploy en Netlify

Configuracion recomendada:

- Build command: `npm run build`
- Framework preset: `Next.js`
- Variable opcional: `NEXT_PUBLIC_SITE_URL=https://szachrajesteticamedica.netlify.app`

El sitio usa imagenes optimizadas en formato WebP y no requiere base de datos, formularios ni funciones serverless.
